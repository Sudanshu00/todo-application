# 📝 Todo Application

A responsive Todo Application built using HTML, CSS, and JavaScript.  
This project allows users to add, view, and delete tasks along with due dates.

---

## 🚀 Features

- ➕ Add new tasks
- 📅 Assign due dates to tasks
- ❌ Delete tasks
- 🎨 Clean and responsive UI
- ⚡ Dynamic updates using JavaScript (DOM manipulation)

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
